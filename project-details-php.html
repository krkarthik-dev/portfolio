Project Overview

This Hospital Management System digitizes hospital operations such as patient registration, appointment scheduling, report generation, and medicine inventory. Supports multiple roles with secure data management.

  • Role-based login for Admin, Doctor, Patient, and Staff
  • Patient registration & profile management
  • Appointment booking and management
  • Medical report & prescription management
  • Medicine stock management
  • Secure data storage and workflow automation

Setup XAMPP:

1. Install XAMPP/WAMP
2. Move project to htdocs: htdocs/HMS/
3. Create database in phpMyAdmin: hospital_db
4. Import database.sql
5. Configure DB connection in connect.php:
   $conn = mysqli_connect("localhost", "root", "", "hospital_db");
6. Open in browser: http://localhost/HMS/
